# WinDownloader - Параллельный загрузчик файлов для Windows

Консольное приложение для параллельной загрузки файлов по списку URL с использованием WinHTTP API. Программа поддерживает HTTP и HTTPS протоколы, работает с заголовком Content-Disposition и обеспечивает многопоточную загрузку.
## 🚀 Возможности

- **Параллельная загрузка** - одновременная загрузка нескольких файлов (настраиваемое количество потоков)
- **Поддержка HTTP/HTTPS** - работает с защищенными соединениями без проверки сертификатов
- **Content-Disposition** - автоматическое извлечение имени файла из заголовков сервера
- **Умное именование** - санитизация недопустимых символов в именах файлов
- **Уникальные имена** - автоматическое добавление (1), (2) при конфликте имен
- **Подробное логирование** - все события с временными метками до миллисекунд
- **Без curl** - полная реализация на WinHTTP API
- **Кроссплатформенность** - сборка для Windows с использованием CMake

## 📦 Требования

### Для сборки:
- Windows 7/8/10/11
- Visual Studio 2017/2019/2022 (Community Edition)
- CMake 3.10 или выше
- MSVC компилятор с поддержкой C++17

### Для запуска:
- Windows 7 и выше
- Права на запись в выходную папку
- Доступ в интернет для загрузки файлов

## 🔧 Сборка проекта

### Способ 1: Visual Studio + CMake (рекомендуемый)

1. **Откройте проект в Visual Studio:**
   - Запустите Visual Studio
   - Выберите **"Открыть локальную папку"**
   - Укажите папку с проектом

2. **Соберите проект:**
   - Нажмите `Ctrl + Shift + B`
   - Или меню **Сборка** → **Собрать все**

3. **Настройте параметры запуска:**
   - Правый клик на `CMakeLists.txt` → **CMake Settings**
   - В разделе **Debugging** укажите **Command Arguments**: `urls.txt C:\downloads 5`

### Способ 2: Командная строка

```bash
# Перейдите в папку проекта

# Создайте папку для сборки
